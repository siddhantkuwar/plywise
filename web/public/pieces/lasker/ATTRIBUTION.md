# Lasker chess pieces

The SVG files in this directory are the **Lasker** chess symbol set by
**Eight Squared Software**.

- Source: https://gitlab.com/eightsquared/lasker
- Pinned source commit: `d63a628e482400e9508d7f0bedbadf095f348829`
- Commit page: https://gitlab.com/eightsquared/lasker/-/commit/d63a628e482400e9508d7f0bedbadf095f348829
- License: Creative Commons Attribution 4.0 International (CC BY 4.0)
- License URL: https://creativecommons.org/licenses/by/4.0/
- Retrieved: 2026-07-19

The twelve files named `white_*.svg` and `black_*.svg` preserve the upstream
SVG markup and geometry from the repository's `dst/svg/` directory. The only
normalization is a trailing newline at end of file. No Chess.com artwork is
included or used as a source.

The complete upstream license text is included in [LICENSE.md](LICENSE.md).
